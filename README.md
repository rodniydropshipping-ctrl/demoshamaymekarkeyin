# דף נחיתה - שמאי מקרקעין מקצועי

דף נחיתה מקצועי יוקרתי למשיא שמאי מקרקעין בעברית עם עיצוב RTL, ולידציה פורם וטיפול lead.

## 📋 תוכן הדף

- **אזור פתיחה (Hero)** - כותרת, תת-כותרת וכפתורי CTA
- **תחומי התמחות** - 6 תחומים עיקריים של עבודה
- **יתרונות** - הצגת נקודות החוזק והניסיון
- **תהליך עבודה** - 4 שלבים של התהליך
- **אזור אמון** - סטטיסטיקות וביזור אמינות
- **טופס לידים** - טופס מלא עם ולידציה
- **כפתור WhatsApp צף** - קישור ישיר לוואטסאפ
- **Footer** - פוטר עם פרטי יצירת קשר

## 🎨 עיצוב

- **צבעים עיקריים**: כחול כהה (#1a2844), כחול בינוני (#2c5aa0), אפור (#2c3e50)
- **רקע**: לבן וגוונת אפור בהיר (#f8fafb)
- **טיפוגרפיה**: Segoe UI | עברית RTL
- **Responsive**: Mobile First - מתאים לכל גודל מסך
- **עיצוב**: יוקרתי, מינימליסטי, סמכותי
